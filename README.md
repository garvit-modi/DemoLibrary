# DemoLibrary

DemoLibrary is a practice Android Gradle library created to understand the complete process of building, publishing, and consuming an Android library.

This project is intended **only for learning and practice purposes**.

---
**Features**

Android Gradle library module
Published and consumed as a Gradle dependency
Demonstrates:
Library creation
Versioning
Dependency integration in Android projects

---

 **Purpose**
 
This library was built for practice to:
Learn Android library development
Understand Gradle dependency publishing
Simulate real-world third-party library usage

## Installation

Add the following dependency to your app-level `build.gradle` file:

```gradle
dependencies {
    implementation 'com.github.garvit-modi:DemoLibrary:v1.0.1'
}
